# side-project-validator

> AI 副业想法验证工具 - 用数据验证你的 idea，而不是听 AI "瞎猜"

## 📌 这是什么？

一个帮助独立开发者、程序员、副业探索者**系统验证副业想法**的 AI Skill。

## 🚀 快速开始

### 验证一个想法

在 OpenClaw 中说：

```
帮我验证一下 [你的想法] 能不能做
```

### 示例

| 输入 | 推荐指数 |
|------|----------|
| 开奶茶店 | ⭐⭐ 谨慎 |
| 程序员副业咨询 | ⭐⭐⭐⭐ 推荐 |
| 殡葬行业信息平台 | ⭐⭐⭐⭐ 可以做 |

## 🎯 Skill 特点

| 特点 | 说明 |
|------|------|
| 数据驱动 | 真实市场数据，不是 AI "直觉" |
| 案例参考 | 50+ 成功/失败案例 |
| 系统验证 | 5 步验证流程 |
| 可操作 | 直接给结论和行动建议 |

## 🆚 vs 直接问 ChatGPT/豆包

| 对比项 | 直接问 AI | 这个 Skill |
|--------|----------|------------|
| 回答方式 | "我觉得可以..." | 数据+评分+结论 |
| 依据 | AI 的直觉 | 真实案例+市场数据 |
| 可操作性 | 泛泛建议 | 具体行动方案 |
