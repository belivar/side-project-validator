# 副业 Idea 验证器

> 用 AI 验证你的副业想法，而不是听 AI "我觉得可以"

## 为什么用这个 Skill？

### 🆚 Skill vs 直接问豆包/ChatGPT

| 对比项 | 直接问 AI | 用这个 Skill |
|--------|----------|-------------|
| **回答方式** | "我觉得可以，你可以试试..." | 数据+案例+评分，给出结论 |
| **依据** | AI 的"直觉" | 真实市场数据、竞品分析、案例库 |
| **深度** | 泛泛而谈 | 5 步系统验证流程 |
| **可操作性** | 建议"你去做调研" | 直接给你调研结果 |
| **案例参考** | 编造的例子 | 真实成功/失败案例库 |

### 📌 核心差异

**豆包说**：
> "奶茶店是个好方向，年轻人喜欢喝，你可以试试加盟蜜雪冰城..."

**Skill 说**：
> 奶茶店需求验证：⭐⭐ 竞争度：极度红海
> - 蜜雪冰城 3 万家门店，霸王茶姬估值 200 亿
> - 60% 奶茶店不赚钱或亏损
> - 你是程序员，没有餐饮经验
> **结论**：开店不如做工具，推荐指数 ⭐⭐

## 这个 Skill 能做什么？

当你有一个副业想法时，帮你：

1. **验证需求真伪** - 是真需求还是伪需求？
2. **分析竞争格局** - 蓝海还是红海？头部玩家是谁？
3. **估算市场规模** - 天花板有多高？
4. **规划变现模式** - 怎么赚钱？能赚多少？
5. **评估实施难度** - 你能做吗？风险在哪？

## 快速示例

### 验证"开奶茶店"

```
输入：帮我看看开奶茶店能不能做

输出：
=== 需求验证 ===
搜索热度高，但供给严重过剩

=== 竞品分析 ===
蜜雪冰城 3 万家、喜茶 3000 家，极度红海

=== 变现分析 ===
投入 20-30 万，回本 2-3 年，60% 亏损

=== 综合评分 ===
推荐指数 ⭐⭐ 谨慎
```

### 验证"程序员副业咨询"

```
输入：我想做程序员副业咨询，能做吗

输出：
=== 需求验证 ===
知乎/B站大量讨论，需求真实

=== 竞品分析 ===
有竞品但可差异化，专注"副业"细分

=== 变现模式 ===
1v1 咨询 500-2000/时，训练营 3000-8000/期

=== 综合评分 ===
推荐指数 ⭐⭐⭐⭐ 推荐
```

## 如何使用

### 方式一：在 OpenClaw 中使用

在 OpenClaw 对话中直接说：

```
帮我验证一下 [你的想法] 能不能做
```

### 方式二：自己部署

```bash
# 克隆仓库
git clone https://github.com/your-repo/side-project-validator.git

# 安装到 OpenClaw
cp -r side-project-validator ~/.qclaw/workspace/skills/
```
